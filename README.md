# AI와 함께 게임 개선하기

## 1. 이 레포는 무엇인가요?

이곳에는 일부러 단순하게 만든 웹 게임이 있습니다. 게임 하나를 골라 자기 AI 에이전트와 대화하며 기능을 빠르게 개선하고, 인터넷에 배포해 실제 사용자의 의견을 받은 뒤 다시 고쳐 보는 실습입니다. 코딩을 몰라도 괜찮습니다. 하고 싶은 일을 평소 말하듯 AI에게 설명해 보세요.

## 2. 게임 받기

다음 두 게임 중 하나를 선택하세요.

- [박스 피하기](box-game/game-ver1.html): 방향키 또는 마우스로 떨어지는 박스를 피합니다.
- [달리기 게임](run-game/game-ver1.html): 스페이스 또는 ↑ 키로 장애물을 뛰어넘습니다.

1. 위 링크를 눌러 파일 페이지를 엽니다.
2. 오른쪽 위의 **Download raw file(다운로드)** 버튼을 눌러 `game-ver1.html` 파일 하나를 저장합니다.
3. 다운로드한 `game-ver1.html`을 더블 클릭합니다. 웹 브라우저에서 게임이 열립니다.
4. 게임이 열리지 않으면 파일을 Chrome, Edge, Safari 같은 브라우저 창으로 끌어다 놓으세요.
5. 원본을 다시 받을 수 있도록, 수정할 파일은 별도 폴더에 복사해 두는 것을 권장합니다.

## 3. AI 에이전트와 개선하기

AI 에이전트에서 `game-ver1.html`이 있는 폴더를 열고, 먼저 이렇게 요청해 보세요.

> 이 폴더의 game-ver1.html을 읽고 게임 구조와 CONFIG 블록을 쉽게 설명해 줘. 아직 파일은 수정하지 마.

파일 위쪽의 `CONFIG` 블록은 게임의 설정 모음입니다. 색상, 이동 속도, 장애물 속도와 생성 간격 같은 값을 한곳에서 바꿀 수 있습니다. 한 번에 한 가지를 요청하고, 수정할 때마다 파일을 브라우저에서 다시 열거나 새로고침해 확인하세요.

쉬운 것부터 차례로 도전해 보세요.

1. **색과 속도 바꾸기**
   > CONFIG 블록에서 플레이어를 노란색으로 바꾸고, 게임 속도를 조금 느리게 해 줘. 무엇을 바꿨는지도 설명해 줘.
2. **최고 점수 저장하기**
   > 브라우저를 닫았다가 다시 열어도 최고 점수가 남도록 만들어 줘. 현재 점수와 최고 점수가 함께 보이게 해 줘.
3. **점점 어려워지게 만들기**
   > 점수가 올라갈수록 장애물이 조금씩 빨라지고 더 자주 나타나게 해 줘. 너무 갑자기 어려워지지 않게 조절해 줘.
4. **새 장애물 만들기**
   > 기존 장애물과 모양이나 움직임이 다른 장애물 종류를 하나 추가해 줘. 기존 게임도 계속 정상 작동하는지 확인해 줘.

**찾아서 고쳐 보기:** 게임을 시작하기 전에 마우스를 움직이면 오류가 나지는 않나요? 또 화면 주사율이 높은 빠른 기기에서 게임이 약 2배 빨라지지는 않나요? AI에게 증상을 설명하고, 원인을 먼저 찾은 뒤 고쳐 달라고 요청해 보세요.

## 4. 배포하기

1. 강사가 공지한 게임 갤러리 **(강사 공지)** 를 엽니다.
2. 화면 위쪽의 **내 게임 업로드** 버튼을 누릅니다.
3. 자신의 **소속(수업)** 을 목록에서 선택하고 **이름**을 입력합니다.
4. 개선한 `game-ver1.html` 파일을 선택합니다.
5. 업로드 버튼을 누릅니다.
6. 완료 화면에 나온 게임 URL을 눌러 실제로 게임이 열리는지 확인합니다.
7. 발급된 URL은 다시 찾을 수 있도록 메모해 두세요. 갤러리로 돌아가면 방금 올린 게임도 확인할 수 있습니다.

## 5. 피드백 받고 다시 개선하기

1. 발급받은 게임 URL을 친구, 가족, 동료에게 공유하고 직접 플레이해 달라고 부탁하세요.
2. “어디가 어려웠나요?”, “무엇을 추가하면 더 재미있을까요?”처럼 구체적으로 물어보세요.
3. 받은 의견을 그대로 AI 에이전트에게 전달하고, 한 가지씩 다시 개선하세요.
4. 수정한 HTML 파일을 업로드 페이지에 다시 올리면 **새 URL이 발급됩니다.** 새 URL이 잘 열리는지 확인한 뒤 다시 공유하세요.

## 6. 업로드 규칙

**업로드할 수 있는 파일은 HTML 파일 1개이며, 크기는 1MB 이하여야 합니다.**
